# springboot-initializr
项目自动生成，并初始化代码于gitlab上

- 需要设置gitlab 的token com.sdcuike.springbootinitializr.util.GitLabApi.privateToken

- 项目使用了org.eclipse.jgit，本地安装git，并设置SSH key
